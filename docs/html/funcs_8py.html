Detailed Description

Module containing functions to define the distributions, prior and posterior for the lighthouse problem, the metropolis-hastings algorithm, and to plot the distributions, and results.

This module contains tools to define the lighthouse problem, and to plot the distributions, and results. It contains 3 functions that define the lighthouse Cauchy distribution, the log likelihood, and the log prior. It also contains a function to run the Metropolis-Hastings algorithm. Finally, it contains functions to plot the lighthouse Cauchy distribution, the lighthouse problem, the corner plot of the chain, the chain of samples, and the Gelman-Rubin statistic.
