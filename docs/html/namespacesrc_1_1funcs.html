Calculate the Gelman-Rubin statistic.

This function takes a number of chains fron the same algorithm started at different initial values, removes the burn-in, and returns the Gelman-Rubin statistic, which is a measure of convergence of the MCMC algorithm, by looking at the variance between the chains and the variance within the chains.

Parameters
chainsThe chains of samples
window_sizeThe size of the window
step_sizeThe step size
ndimThe number of dimensions
Returns
The Gelman-Rubin statistic
